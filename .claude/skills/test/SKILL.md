テストを実行してください。

対象リポジトリ: https://github.com/nikawa2161/blog

## 実行手順

### 1. テストの実行

`bun test` コマンドを使用してテストを実行:

```bash
bun test
```

### 2. テスト結果の確認

テスト実行結果を確認し、以下を報告:
- 成功したテストの数
- 失敗したテストの数
- テスト実行時間

### 3. テスト失敗時の対応

テストが失敗した場合:
- エラーメッセージとスタックトレースを表示
- 失敗原因を分析
- 修正方法を提案

## 自動提案のタイミング

以下のタイミングでテスト実行を提案します:
- コード変更完了後
- コミット前
- PR作成前
- ビルド実行前

## 注意事項

- テストが失敗した場合は、必ず修正してから次の作業に進む
- テストカバレッジが低下していないか確認
- 新しいコードには適切なテストを追加
